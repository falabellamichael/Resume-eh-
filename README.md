# Resume

Personal resume website for Michael Anthony Falabella.

## Files

- `index.html` — the resume webpage. GitHub Pages uses this as the homepage.
- `.nojekyll` — tells GitHub Pages to publish the static HTML directly.

## Publish on GitHub Pages

1. Create a GitHub repository named `Resume`.
2. Upload `index.html`, `.nojekyll`, and this `README.md` to the repository root.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select branch: `main` and folder: `/ (root)`.
6. Save.

The website should be available at:

```text
https://falabellamichael.github.io/Resume/
```

If your GitHub username is different, replace `falabellamichael` in the URL with your actual username.
